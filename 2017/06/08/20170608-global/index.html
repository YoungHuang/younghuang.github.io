彭博新闻社：中国的全球科技强国之路从东南亚启程

Posted on 六月 8, 2017 本文总阅读量次

Isaac Ho与一些投资者在新加坡观看一家癌症检测初创公司的演示，当时一位在场的中国亿万富豪站起身，默不作声地在白板上写下几个潦草的中文字“速度x市场份额”。这个简单公式的含义是，不惜一切代价成为首个、同时也是最大的一个。但Ho就是在那时知道东南亚的科技格局即将发生改变。

这位Venturecraft Group的创立人说，那一刻我懂得了中国的策略。他因常常举办觥筹交错的威士忌派对在本地医疗技术界为人熟知。他表示，如果你做不到第一，就会被淘汰；只有成为第一，才能购买更先进的技术；它是一场胜者为王的游戏。

阿里巴巴集团、腾讯和滴滴出行通过抢占市场成为行业巨头，其成长的速度与规模让许多人惊诧：如今阿里巴巴与腾讯位居全球前10大公司之列。成立五年的滴滴出行击败Uber Technologies Inc.的一个手段就是烧钱。眼下，由于国内市场增速放缓并且已在饱和，为寻找新的增长来源，中国的科技业霸主把目光转向海外市场。第一站是东南亚：该地区总人口是美国的两倍，还是全球华裔最多的地方。

据普华永道估计，去年中国对海外科技业的投资增长逾一倍，达到378亿美元。其中，阿里巴巴2016年出资10亿美元取得了新加坡电商公司Lazada Group SA的控股权，目前该公司已成为阿里巴巴在东南亚的桥头堡。已参股东南亚估值最高初创企业Sea Ltd.的腾讯，据悉接近达成对印尼共享出行服务公司Go-Jek的投资。甚至亚洲估值最高的初创公司滴滴出行，也投资了东南亚的同行Grab，并宣布将走向全球。

戈壁的创始合伙人曹嘉泰说，你所目睹的是思维方式的改变。他们期盼的不仅仅是成为中国最大公司，而且想成为全球的行业翘楚。

中国数十年来，为担当地区经济的掌门人奠定了基础。国内富有的投资者向从交通运输到房地产的各个领域投下数十亿计的资金，他们的投资正在改变这个地区。瑞信估计，仅2016年，中国对东南亚最大六个国家的外商直接投资几乎翻了一番。

起初，中国的投资很少涉及科技业。但随着移动设备的更广泛使用以及中产阶级的崛起，中国的科技大公司开始重视这个产业。东南亚地区生活着全球最多的华裔人口，这让渴求文化共性的未来金融家们感到安心。国际货币基金组织(IMF)预测，东盟的五个成员国–印尼、马来西亚、菲律宾、泰国和越南到2022年的经济年增速将超过5%，北亚国家的平均年增速料为3%。

这个地区仍然还存在可以抢占市场的机会。Grab与Go-Jek竞争共享出行服务市场，Tokopedia与Lazada在电子商务市场展开激战，但尚无一家公司在任何一个领域取得掌控权。相比之下，中国少数几家公司已主宰了一些重要领域，如百度的搜索业务，阿里巴巴的电子商务、腾讯的社交媒体，以及滴滴出行的共享出行服务。

以色列科技界先驱Yossi Vardi表示，阿里巴巴积极攻陷海外市场的举动令其回想起上世纪60-70年代的美国，那时美国公司开始在海外寻找增长机会，最终成为跨国企业。他上月在新加坡一个会议上说，“这点非常重要，而且才刚刚开始。”
